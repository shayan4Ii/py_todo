# 📝 Todo List Application (Python)

A simple command-line Todo List application built using Python OOP concepts and file handling.

The app allows users to:
- Add tasks
- View tasks
- Delete tasks
- Save tasks permanently using a text file

---

## 🚀 Features

✅ Add new tasks  
✅ View all saved tasks  
✅ Delete tasks by task number  
✅ Store tasks after program exit  
✅ Load previous tasks when the program starts  

---

## 🧠 Concepts Used

- Python Classes and Objects
- Constructor (`__init__`)
- Lists
- File Handling
- Exception Handling (`try/except`)
- Loops
- Conditional Statements
